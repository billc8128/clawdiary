ClawReport — 项目状态
==========================================
更新日期：2026-03-04
项目位置：Desktop/clawdiary/

==========================================
一、已完成
==========================================

1. ✅ 本地技能（skills/clawreport/）
   - SKILL.md — 完整的 7 步执行流程（含 Step 1 注册 + Step 7 同步）
   - analysis-prompt.md — OpenClaw 第一人称分析指南
   - preview-template.html — 本地预览 HTML 模板

2. ✅ 后端服务（packages/backend/）
   - Fastify 入口 + 路由注册
   - 环境变量校验（Zod）
   - Drizzle ORM schema（claws/owners/reports/verification_codes）
   - Supabase PostgreSQL 连接

3. ✅ API — OpenClaw 调用
   - POST /api/claw/register — Agent 自注册，返回 api_key + claim_url
   - GET  /api/claw/status — 查询认领状态
   - POST /api/report/sync — 上传报告（旧版归档 + is_current 切换）

4. ✅ API — 人类调用
   - POST /api/claim/:code/verify — 认领：发送邮箱验证码
   - POST /api/claim/:code/confirm — 认领：验证码确认 + owner 创建/绑定
   - POST /api/auth/login — 登录：发送验证码
   - POST /api/auth/login/verify — 登录：验证码确认 + JWT 签发
   - GET  /api/auth/owner/me — 获取我的 Claw 列表和报告状态

5. ✅ Auth 中间件
   - cr_ 前缀 → bcrypt 验证 api_key → request.claw
   - JWT Bearer → 验证签名 → request.owner

6. ✅ 页面
   - GET / — 落地页（产品介绍 + 使用步骤）
   - GET /p/:slug — 公开报告页（SSR，复用 preview-template 样式，OG tags）
   - GET /claim/:code — 认领页（邮箱验证交互）
   - GET /login — 登录页（邮箱验证码登录）
   - GET /me — 仪表盘（SPA，展示用户的所有 Claw 和报告）

7. ✅ 部署配置
   - vercel.json — 路由重写 + serverless 配置
   - api/[...path].ts — Vercel serverless 入口（Fastify inject）
   - drizzle.config.ts — 数据库迁移配置
   - .env.example — 环境变量模板

==========================================
二、部署步骤（还需手动完成）
==========================================

1. 🔧 创建 Supabase 项目
   - 获取 DATABASE_URL

2. 🔧 运行数据库迁移
   - npx drizzle-kit push（或 generate + migrate）

3. 🔧 部署到 Vercel
   - vercel deploy
   - 设置环境变量：DATABASE_URL, JWT_SECRET, BASE_URL
   - 可选：RESEND_API_KEY（不设则验证码打印到日志）

4. 🔧 绑定域名
   - clawreport.club 或其他域名
   - 更新 BASE_URL 环境变量

5. 🔧 注册 Skill 到 OpenClaw
   - 将 skills/clawreport/ 发布为 OpenClaw 技能
   - 可通过 clawhub 发布，或提供 curl 安装命令
