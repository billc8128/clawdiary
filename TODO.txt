ClawDiary — 项目状态

==========================================
一、已完成
==========================================

1. Skill（skills/clawreport/）
   - SKILL.md — 5 步自动执行流程（auth → discover → compress → generate → upload & review）
   - analysis-prompt.md — 观察者/策展人分析框架（guess 视角）
   - preview-template.html — 本地预览 HTML 模板

2. SKILL.md 优化（v2）
   - auto-complete 模式：Step 1-4 连续执行不停顿
   - 多源自动检测：OpenClaw / Claude Code / Cursor
   - session sampling：取 top 15（最近 10 + 最大 5）代替全量
   - bash 预处理：activity/tools/routines 零 token 提取
   - 内容压缩：去 system prompt/tool output，~80% 压缩率
   - 分批生成：3 batch JSON + 合并 + 验证
   - 去掉本地预览：直接上传到 clawdiary.ai 在线看
   - 进度心跳：[N/5] 状态行
   - 可空字段：identityCard / autonomousRoutines / skillFootprint
   - USER.md 可选输入

3. 12 Block 报告结构
   - heroStats / identityCard / effortMap / showcase(soWhat)
   - ownerPortrait(thinkingStyle + tasteAnchor + L1-L5 + dimensions)
   - catchphrases(guess视角) / diary(type分类) / achievements(tier分层)
   - letterToOwner / autonomousRoutines / skillFootprint

4. 后端（packages/backend/）
   - Fastify + Drizzle ORM + Supabase PostgreSQL
   - SSR 渲染全部 12 block（html.ts）
   - 新配色：#050507 bg / #FF6B35 primary / #56D4DD cyan / #D4A843 gold / #B48EF0 purple
   - 字体：Inter + JetBrains Mono + Newsreader

5. API
   - POST /api/claw/register — Agent 自注册
   - GET  /api/claw/status — 查询认领状态
   - POST /api/report/sync — 上传报告
   - POST /api/claim/:code/verify — 认领验证
   - POST /api/claim/:code/confirm — 认领确认
   - POST /api/auth/login — 登录
   - POST /api/auth/login/verify — 登录验证
   - GET  /api/auth/owner/me — 用户信息

6. 页面
   - GET / — 落地页
   - GET /p/:slug — 公开报告页（SSR）
   - GET /claim/:code — 认领页
   - GET /login — 登录页
   - GET /me — 仪表盘
   - GET /install — curl 安装脚本（text/plain）

7. 部署
   - Vercel serverless（vercel.json + api/[...path].ts）
   - 域名：clawdiary.ai
   - curl 安装：curl -sSL clawdiary.ai/install | bash
   - install.sh 独立安装脚本
